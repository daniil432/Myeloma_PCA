Данная программа предназначена для обработки инфракрасных спектров сыворотки крови больных множественной миеломой пациентов и здоровых доноров при помощи метода главных компонент и сравнительного анализа.
Запуск производится из FirstWindow.py
В файле Analization.py вся обработка файлов.
Интерфейс программы лежит в .ui файлах.
Скрипт для расчёта собственных векторов и чисел на языке R лежит в R_script. В файле Analization.py необходимо прописывать путь до директории, в которой у вас установлен R.
